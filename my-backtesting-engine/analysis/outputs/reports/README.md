# Reports Output Folder

Save analysis reports here.

## Types of Reports

1. **Text Reports** (`.txt`, `.md`)
   - Analysis summaries
   - Findings documentation
   - Performance comparisons

2. **CSV Reports** (`.csv`)
   - Screening results
   - Performance metrics
   - Statistical summaries

3. **Excel Reports** (`.xlsx`)
   - Multi-sheet analyses
   - Formatted tables
   - Combined data

## Naming Convention

- `{analysis_type}_{date}.{ext}`
- Example: `sector_performance_2026_02.xlsx`

## Best Practices

- Include date in filename
- Add README or notes in report
- Keep raw data separate from summaries
- Version control important reports
