# Charts Output Folder

Save all generated charts and visualizations here.

## Naming Convention

Use descriptive filenames:
- `{stock_symbol}_{indicator}_{date}.png`
- Example: `RELIANCE_rsi_2026_02_01.png`

## Organization

Optionally create subfolders:
- `technical/` - Technical analysis charts
- `fundamental/` - Fundamental analysis charts
- `comparison/` - Multi-stock comparisons
- `equity_curves/` - Backtest results

## Tips

- Use high DPI (300) for publication quality
- Include dates in titles
- Save with descriptive names
- Clean up old files periodically
